Experience Chat-Time:

A Full-Stack Chat application developed using Node, Express and MongoDB to provide a secure and a personalized experience when chatting with your loved ones.

Features.

Features.

  • Consists of an OAuth User-Authentication Framework to help store a client's personal details for Logging and Signing-In.

  • User-details are encrypted using Hashing Algorithms and are not reachable to the developer of the application.

What's Next?

What's Next?

    Enabling Video and Audio Calls over the Internet.

    Creating private and secure Rooms which shall allow clients to seperately communicate with each other.

    Integrating OAuth2 to improve security due to the availability of short-lived access tokens.
Back-End.

Hover to understand the Back-End Frameworks used for Chat-Time.

NodeJS and MongoDB.

Tools for developing the Back-End.

  • Used NodeJS to develop the server-side of the application.
  • MongoDB was used to store user the Hashed User-Credentials.
  • Utilized the Websocket API to establish a connection between the server and multiple clients. The WebsocketClient and the WebsocketServer provided both client and server functionality.
Front-End.

HTML, CSS3 and Express.

Tools for developing the Front-End.

  • Utilized HTML and CSS3 to control the layout and the properties of all the elements on the webpage.
  • Express was also used to structure the application to handle multiple HTTP requests at specific URL's.
  • Combining the above frameworks helped to develop the application in a secure and a modern way.
